k14s

Kubernetes Tools that follow Unix philosophy to be simple, modular, and extensible

$ ytt tpl -R -f . | kbld -f - | kapp -y deploy -a app1 -f -

github.com/k14s collection: ytt | kbld | kapp | kwt

Install

On Linux:

$ wget -O- https://k14s.io/install.sh | bash

On OS X (based on github.com/k14s/homebrew-tap):

$ brew tap k14s/tap
$ brew install ytt
$ brew install kbld
$ brew install kapp
$ brew install kwt
